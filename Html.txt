---  📌 1.2 Common HTML Attributes  Attribute	Usage Example	Notes

id	<div id="main">	Unique identifier
class	<p class="text">	CSS selector group
style	<h1 style="color:red;">	Inline CSS
src	<img src="img.png">	For images, scripts
href	<a href="url">	Hyperlinks
alt	<img alt="desc">	Image description
title	<div title="tooltip">	Tooltip text
value	<input value="data">	Input default
type	<input type="text">	Input type (text, email)
placeholder	<input placeholder="Enter...">	Grey hint text
disabled	<input disabled>	Makes input inactive
checked	<input checked>	Checkbox default
required	<input required>	Must fill before submit
target	<a target="_blank">	Open in new tab
rel	<link rel="stylesheet">	Link relationship

🎨 PART 2: CSS (Styling)

📌 2.1 CSS Syntax

selector {
property: value;
}

📌 2.2 Common Selectors

Selector	Meaning

Universal
div	Element
.className	Class
#idName	ID
div p	Descendant
div > p	Direct child
div + p	Next sibling

📌 2.3 Most Used Properties

Text & Font

color, font-size, font-family, text-align, text-decoration

Box Model

width, height, padding, margin, border, box-sizing

Background

background-color, background-image

Layout

display: flex | grid | block | inline

position: static | relative | absolute | fixed

top, left, right, bottom, z-index

Others

opacity, visibility, overflow, transition, transform

📌 2.4 Pseudo-Classes & Pseudo-Elements

a:hover      /* Mouse over /
input:focus  / Input focused */
p::first-letter { font-size: 2em; }

📌 2.5 Media Queries (Responsive Design)

@media screen and (max-width: 768px) {
body { font-size: 14px; }
}

---

✅ PART 5: Best Practices

Use semantic HTML (<section>, <article>, etc.)

Use alt for all images

Use external CSS and JS files

Validate your HTML & CSS

Mobile-first design (media queries)

Minify CSS/JS in production

Keep JavaScript in bottom of body or use defer

Use responsive units (em, %, vh)
